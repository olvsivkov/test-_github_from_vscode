Старый анекдот про программиста переведенный в код

Жена говорит программисту:
- Дорогой, сходи, купи булки. И если есть яйца, возьми десяток.
Программист приходит в магазин:
- Яйца есть?
- Есть.
- Дайте десять булок.

function bred() {
  let eggs = prompt("how many eggs in a shop?",0);
   if (eggs >= 1){
    return ("купил" + " " + (1*10) + " " + "булок");
   } else {
    alert("no eggs in the shop");
   }
  };
 alert(bred());
